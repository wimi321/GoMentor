# Josekle dictionary SGF

This directory bundles `dictionary.sgf` from Josekle:

- Source: https://github.com/okonomichiyaki/josekle/tree/master/sgf
- Repository license: MIT, see `LICENSE` in this directory.

The SGF is loaded as a variation tree and converted into joseki-family recognition cards. GoMentor does not quote the original comments to the LLM teacher; it only uses normalized move prefixes and next-move candidates for recognition.
