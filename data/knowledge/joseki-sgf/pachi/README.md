# Pachi joseki SGF set

This directory bundles the joseki SGF files from the Pachi project:

- Source: https://github.com/pasky/pachi/tree/master/joseki
- License: GPL-2.0-only, see `COPYING` in this directory.
- Files: `joseki_33.sgf`, `joseki_34.sgf`, `joseki_44.sgf`, `joseki_54.sgf`.

GoMentor loads these SGF variation trees at runtime and converts prefixes into joseki-family recognition cards. Keep this directory's license notice when redistributing packaged builds.
